# Enseignements
Mes enseignements se font principalement au niveau L3 (Algorithmique et programmation, Programmation orientée objet) pour des ingénieurs généralistes, et dans des modules de spécialité ensuite (bases des Data Sciences, Programmation Mobile sur iOS en Swift) en numérique et industrie et service.

Enseignements déjà réalisés :
- Langage C
- Introduction à l'algorithmique, introduction à Java
- Programmation orientée objet en Java
- Bases de traitement et analyse de données (sélection de variables, présentations des algorithmes de classifications "classiques")
- Analyse d'image et deep learning
- Bases du traitement du signal pour les données physiologiques
- Développement matériel et logiciel pour des mesures de données physiologiques sur l'humain

Tout ceci s'accompagne de plusieurs projets proposés aux étudiants.

Avec le pôle ingénierie pédagogique de l'IMT Nord Europe, nous avons aussi développé un escape game avec les bases de l'informatique, ou encore nous travaillons sur un jeu sérieux pour apprendre les bases de Java (avec l'université de Grenoble et l'université de Genève).

Enfin, nous travaillons sur des solutions permettant d'évaluer les étudiants différemment. Ainsi, nous avons travaillé sur des challenges qui se basent sur la réalisation de stratégie pour des jeux de société afin de noter automatiquement les étudiants (avec des paliers donnés par des stratégies enseignants et des vérifications de plagiat). Cela nous permet de noter des compétences différentes de nos étudiants en cycle ingénieur.
